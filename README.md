# Codex Frontend Craft

Super skill de design system para front-end no GPT-5 Codex. Orienta a criacao de interfaces com foco em direcao visual clara, tipografia expressiva, layout intencional e execucao de alto nivel.

## Descricao
Projeto que define regras, criterios e fluxo de trabalho para gerar UI/UX de qualidade, evitando resultados genericos.

## O que inclui
- `skill.yaml` com metadados da skill
- `system.md` com o guia principal de design

## Como usar
- Adicione a skill ao Codex e invoque nas suas solicitacoes de UI.
